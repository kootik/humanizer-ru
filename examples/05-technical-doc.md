> # Установка И Настройка Утилиты
>
> Это раздел про установку.
>
> Данная утилита представляет собой инструмент командной строки для работы с Git, который не только ускоряет рутинные операции, но и обеспечивает стандартизацию процессов в команде разработчиков. Установка утилиты осуществляется посредством менеджера пакетов npm.
>
> ## Системные Требования
>
> Перед началом установки необходимо удостовериться в наличии следующего:
>
> - **Node.js:** Должен быть установлен Node.js версии не ниже 18.
> - **npm:** Должен быть доступен npm версии 8 или выше.
> - **Git:** Должен быть установлен Git версии 2.30 или выше.
>
> ## Процесс Установки
>
> Давайте рассмотрим процесс установки. Вот что вам нужно знать.
>
> Установка осуществляется путём выполнения следующей команды:
>
> ```bash
> npm install -g git-helper
> ```
>
> После завершения установки конфигурационный файл не требуется, так как настройки сохраняются автоматически.
>
> По сути, главный вопрос в том, как начать использовать утилиту. Запустите `git-helper --help` для получения списка доступных команд.
