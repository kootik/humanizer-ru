# Примеры редактуры

Готовые примеры «До → После» на разных типах текстов. Используются как образцы стиля - LLM ориентируется на ближайший по типу пример при работе с похожим текстом.

## Содержимое

| Файл | Тип текста | AI-Slop Score до → после |
|------|------------|--------------------------|
| [01-marketing.md](01-marketing.md) | Маркетинговая страница | 95 → 12 |
| [02-news-article.md](02-news-article.md) | Новостная статья | 65 → 18 |
| [03-business-letter.md](03-business-letter.md) | Деловое письмо (email) | 58 → 14 |
| [04-blog-post.md](04-blog-post.md) | Пост в блоге (с голосом) | 67 → 11 |
| [05-technical-doc.md](05-technical-doc.md) | Технический README | 42 → 8 |

## Как использовать

При работе с новым текстом скилл сначала классифицирует его (маркетинг/новость/техдок/деловое/соцсети), потом смотрит на ближайший пример как на образец стиля и тона.

## Принцип примеров

Примеры **специально не идеальны** - в каждом «После» есть неровности, разговорные обороты, минимум одна неправильность. Это сделано намеренно: идеально чистый текст сам по себе выглядит как AI-выход. Живой текст имеет шероховатости.
