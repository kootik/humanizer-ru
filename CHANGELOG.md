# Changelog

Все значимые изменения в этом проекте документируются здесь.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [2.6.0-ru] - 2026-04-17

Strengthening release. Применены рекомендации `skill-architect-ultra` уровней 1-2: добавлены измеримость, тестируемость, автоматизация и эталонные примеры. Скилл прошёл с B-/C+ (3.45) до A-/A (4.4+).

### Добавлено

- **AI-Slop Score** - числовая оценка зашумлённости текста (0-100) с формулой `Score = min(100, round(100 × Σ(weight × count) / max(words/10, 1)))`. Веса по группам: A=4, B=3, C=2, D=1. Score нормализован на длину текста, поэтому сопоставим для коротких и длинных текстов.
- **`scripts/scan_markers.py`** - Python 3.8+ скрипт без зависимостей. Regex-сканер по словарю маркеров. Запускается ПЕРЕД LLM-проходом для предварительной разметки. Поддерживает stdin, JSON-вывод, quiet-режим. Exit codes: 0 (чистый) / 1 (нужна правка) / 2 (сильный слоп) / 3 (ошибка).
- **`references/test-cases.md`** - 12 эталонных тест-кейсов разных типов (маркетинг, новость, деловое, юридическое, техдок, личное, пресс-релиз, соцсети, чистый текст, D7-кейс, A1 хвостовой, не-русский). Сводная таблица для регрессии.
- **`references/scoring-rubric.md`** - шкала оценки качества редактуры (Edit Quality Score). 5 измерений × 5 баллов: Accuracy (30%), PatternRemoval (25%), Naturalness (20%), Appropriateness (15%), VoicePreservation (10%). Минимальные пороги: Accuracy ≥4, PatternRemoval ≥3, Appropriateness ≥3.
- **`examples/`** - папка с 5 готовыми примерами До/После разных типов текста, каждый с метриками и Edit Quality Score.
- **Edge cases в SKILL.md** - обработка не-русского текста, очень короткого (<5 слов), очень длинного (>10000 слов), Markdown-разметки, битой кодировки, художественного текста (отказ).
- **`allowed-tools`** в frontmatter - Read, Write, Edit, Glob, Grep, Bash. Автопредодобрение прав в Claude Code.

### Изменено

- **Frontmatter**: усилены negative triggers (теперь явно отказывает на «исправь опечатки», «проверь грамматику», «переведи»). Добавлен `version: 2.6.0-ru`. Минимальная длина текста - от 1 предложения / 5 слов.
- **Режим 1 (полная редактура)** - формат вывода теперь включает блок «Метрики»: AI-Slop Score было/стало, число паттернов по группам, покрытие %, изменение объёма.
- **Режим 2 (аудит)** - добавлена шкала AI-Slop Score 0-100 с интерпретацией. Аудит теперь даёт numeric baseline, а не только перечень.
- **Шаг 5 (Верификация)** - чеклист расширен на 4 пункта (B7, C7, D6, D7).
- **Контекст в конце SKILL.md** - таблица всех файлов с назначением, инструкция по запуску `scan_markers.py`.

## [2.5.1-ru] - 2026-04-16

Синхронизация с upstream [blader/humanizer v2.5.1](https://github.com/blader/humanizer) (версии 2.2.0-2.5.1).

### Добавлено

- **B7**: Безличные и бессубъектные конструкции - аналог passive voice (upstream #80). Пример: «данные сохраняются автоматически» → «система сохраняет данные».
- **C7**: Авторитетные тропы - «по сути», «на самом деле», «главный вопрос в том», «по большому счёту» (upstream #79).
- **D6**: Сигнальные анонсы - «давайте разберём», «перейдём к сути», «вот что нужно знать» (upstream #79).
- **D7**: Раздробленные заголовки - заголовок + пустая фраза-повтор (upstream #79).
- Раздел **«Калибровка голоса»** в SKILL.md - работа с образцом стиля пользователя (upstream #64).
- **Шаг 6 «Анти-AI проход»** в процесс редактуры.
- `version: 2.5.1-ru` в frontmatter SKILL.md.
- CHANGELOG.md для истории изменений.

### Изменено

- **A1** расширен: добавлены хвостовые отрицания (tailing negations) - «…, без догадок», «…, без лишних движений» (upstream #79).
- `word-scanner.md`: добавлены маркеры во все группы + расширенный англоязычный AI-слоп.
- Чеклист верификации в Шаге 5: новые пункты для B7/C7/D6/D7.
- Полный пример в SKILL.md переделан - теперь покрывает все 26 паттернов (было 7).
- README.md: обновлены таблицы паттернов (21 → 26), добавлена секция «Калибровка под ваш стиль».

## [2.1.0-ru] - 2026-02-11

Первый публичный релиз скилла для русского языка.

### Добавлено

- 21 паттерн AI-слопа, адаптированных для русского (канцелярит, отглагольные существительные, цепочки родительных падежей, авторитетный тон через «является/представляет собой» и др.).
- 3 режима работы: полная редактура, аудит, точечная правка.
- Классификация текста по типу с разной степенью вмешательства.
- Справочник паттернов (`references/patterns.md`).
- Быстрый сканер слов-маркеров (`references/word-scanner.md`).
- Поддержка смешанных RU+EN текстов (англоязычный AI-слоп как маркеры).
